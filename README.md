# StreetArt Guide - Цифровой путеводитель по уличному искусству

Интерактивный веб-сайт для навигации по уличному искусству города с галереей работ, планировщиком визита, базой художников, образовательным квизом и системой избранного.

## Функциональность

### Главная страница (Галерея)
- **Интерактивная галерея** из 30 арт-объектов с детальной информацией
- **Фильтрация** по районам, типам (Граффити, Мурал, Стенсил, Мозаика, 3D-арт), стилям (Реализм, Абстракция, Портрет, Леттеринг, Характер) и художникам
- **Сортировка** по дате (новые первыми), алфавиту и случайному порядку
- **Модальные окна** с полной информацией об объектах, галереей фотографий, описанием и кнопками действий

### Планировщик визита
- **Автоматический расчет** времени и расстояния тура
- **Добавление/удаление** объектов из плана
- **Изменение порядка** объектов (кнопки "Вверх/Вниз")
- **Визуализация** районов и статистики плана
- **Сохранение планов** в localStorage
- **Готовые планы** (3 предустановленных маршрута)
- **Предупреждения** о расположении объектов

### База художников
- **Профили 10 художников** с биографиями, цитатами и фотографиями
- **Галерея лучших работ** каждого художника (6 работ)
- **Фильтрация** по стилю и поиск по имени
- **Кнопка "Добавить все работы в план"** для быстрого планирования

### Интерактивный квиз "Угадай стиль"
- **10 случайных вопросов** из базы из 15
- **Крупные изображения** арт-объектов
- **Объяснения** правильных ответов
- **Система оценок**: Эксперт, Отличное знание, Неплохо, Есть куда расти, Начало пути
- **Поделиться результатом** в социальных сетях

### Система избранного
- **Добавление в избранное** одним кликом (кнопка-сердечко)
- **Отдельная страница** со всеми избранными объектами
- **Кнопка "Добавить все в план"** для быстрого планирования
- **Сохранение в localStorage**

## Технологии

- **HTML5** с семантической разметкой
- **CSS3** с БЭМ методологией
- **Vanilla JavaScript** (ES6+)
- **localStorage** для сохранения данных
- **Адаптивный дизайн** (desktop, tablet, mobile)

## Технические детали

- **Методология CSS**: БЭМ (Block Element Modifier)
- **Хранение данных**: localStorage (план, избранное, сохраненные планы)
- **Загрузка изображений**: Lazy loading для оптимизации
- **Доступность**: ARIA-атрибуты, семантическая разметка, alt-тексты
